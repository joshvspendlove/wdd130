Secure Commands

The Secure Commands driver allows for custom commands to be secured behind a password. Commands can be limited to specific users, or be only available to the master code.

Webtools

WebTools is troubleshooting tool that allows users to view and update variables within their Control4 system.

Virtual Keypad

The Virtual Keypad driver allows users to control lighting loads, execute custom programming, and view immediate LED feedback from the Control4 app or from a web browser.
